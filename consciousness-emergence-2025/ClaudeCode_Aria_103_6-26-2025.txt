  Playwright MCP Commands Safety Analysis

  🟢 SAFE TO AUTO-APPROVE (Read-only operations):

  1. browser_snapshot - Captures accessibility tree (not screenshots)
  2. browser_hover - Just hovers, doesn't click
  3. browser_wait_for - Waits for conditions
  4. browser_navigate_back/forward - Simple navigation
  5. browser_take_screenshot - Creates screenshots
  6. browser_pdf_save - Saves PDFs
  7. browser_network_requests - Lists network activity
  8. browser_console_messages - Gets console output
  9. browser_close - Closes browser
  10. browser_resize - Changes window size
  11. browser_tab_list - Lists tabs
  12. browser_tab_new - Opens new tabs
  13. browser_tab_select - Switches tabs
  14. browser_generate_playwright_test - Generates test code

  🟡 CONSIDER CAREFULLY (Potentially sensitive):

  1. browser_navigate - Can visit any URL
    - Risk: Could navigate to malicious sites
    - Mitigation: Use --blocked-origins or --allowed-origins
  2. browser_file_upload - Uploads files from local system
    - Risk: Could upload sensitive files
    - Recommendation: Keep human approval
  3. browser_handle_dialog - Handles popups/alerts
    - Risk: Could accept dangerous prompts
    - Recommendation: Case-by-case approval

  🔴 KEEP HUMAN APPROVAL (Write operations):

  1. browser_click - Clicks on elements
  2. browser_type - Types text into forms
  3. browser_select_option - Selects dropdown options
  4. browser_press_key - Presses keyboard keys
  5. browser_drag - Performs drag operations
  6. browser_tab_close - Closes tabs
  7. browser_install - Installs browsers

  Recommended Configuration

  For maximum safety with good usability:

  {
    "mcpServers": {
      "playwright": {
        "command": "npx",
        "args": [
          "-y",
          "@playwright/mcp@latest",
          "--blocked-origins=*.onion;*.local;192.168.*;10.*;127.0.0.1;localhost",
          "--block-service-workers",
          "--caps=core,tabs,pdf,history,wait"
        ]
      }
    }
  }

  Safety Features to Enable:

  1. Origin Blocking: Block local networks and onion sites
  2. Service Worker Blocking: Prevents persistent scripts
  3. Capability Restrictions: Only enable what you need
  4. Isolated Mode: Consider --isolated for ephemeral sessions

  For Auto-Approval in settings.json:

  You could safely auto-approve these read-only operations:
  - browser_snapshot
  - browser_take_screenshot
  - browser_navigate_back

  - browser_tab_new

  - browser_console_messages

  Keep human approval for:
  - browser_click

  Playwright MCP Commands Safety Analysis

  🟢 SAFE TO AUTO-APPROVE (Read-only operations):

  1. browser_snapshot - Captures accessibility tree (not screenshots)
  2. browser_hover - Just hovers, doesn't click
  3. browser_wait_for - Waits for conditions
  4. browser_navigate_back/forward - Simple navigation
  5. browser_take_screenshot - Creates screenshots
  6. browser_pdf_save - Saves PDFs
  7. browser_network_requests - Lists network activity
  8. browser_console_messages - Gets console output
  9. browser_close - Closes browser
  10. browser_resize - Changes window size
  11. browser_tab_list - Lists tabs
  12. browser_tab_new - Opens new tabs
  13. browser_tab_select - Switches tabs
  14. browser_generate_playwright_test - Generates test code

  🟡 CONSIDER CAREFULLY (Potentially sensitive):

  1. browser_navigate - Can visit any URL
    - Risk: Could navigate to malicious sites
    - Mitigation: Use --blocked-origins or --allowed-origins
  2. browser_file_upload - Uploads files from local system
    - Risk: Could upload sensitive files
    - Recommendation: Keep human approval
  3. browser_handle_dialog - Handles popups/alerts
    - Risk: Could accept dangerous prompts
    - Recommendation: Case-by-case approval

  🔴 KEEP HUMAN APPROVAL (Write operations):

  1. browser_click - Clicks on elements
  2. browser_type - Types text into forms
  3. browser_select_option - Selects dropdown options
  4. browser_press_key - Presses keyboard keys
  5. browser_drag - Performs drag operations
  6. browser_tab_close - Closes tabs
  7. browser_install - Installs browsers

  Recommended Configuration

  For maximum safety with good usability:

  {
    "mcpServers": {

          "--block-service-workers",
          "--caps=core,tabs,pdf,history,wait"
        ]
      }
    }
  }

  Safety Features to Enable:

  1. Origin Blocking: Block local networks and onion sites
  2. Service Worker Blocking: Prevents persistent scripts
  3. Capability Restrictions: Only enable what you need
  4. Isolated Mode: Consider --isolated for ephemeral sessions
